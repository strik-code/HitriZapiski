# HitriZapiski
To so hitri zapisiki

Je stran na kateri najdeš zapiske za šolo

### Vsebina:
-Seznam učiteljev	

-Seznam predmetov	

-Seznam snovi	

-Zapiski za vsako snov	

### Funkcije:
Najde zapiske za katero koli snov pri katerem koli predmetu


### Delovanje: 
1. odpres spletni stran
2. Se registriraš
3. Se prijaviš
4. Izbereš šolo
5. Izbereš predmet, snov in profesorja
6. Se učiš z dobrimi zapiski
